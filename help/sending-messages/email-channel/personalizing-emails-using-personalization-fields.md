# パーソナライゼーションフィールドを使用した E メールのパーソナライズ

パーソナライゼーションフィールドは、配信されるメッセージのコンテンツをパーソナライズする第 1 レベルの手段として使用されます。メインコンテンツに挿入したフィールドが、選択したデータソースからのデータの挿入位置を示します。
