# ターゲティングワークフローの作成

Adobe Campaignでは、ワークフローを使用して、ターゲティングキャンペーンの実行、キャンペーンの作成、またはAdobe Campaignの技術プロセスの実行をおこないます。 このビデオでは、ターゲティングワークフローに焦点を当てます。 ターゲティングワークフローにより、配信ターゲットをいくつか作成できます。
