# 受信者リストの作成

リストは、配信アクションのターゲットにしたり、インポート操作時またはワークフロー実行時に更新したりできる受信者の静的なセットです。受信者リストはオーディエンスとも呼ばれます。

エクスプローラーから受信者のリストを設定してオーディエンスを作成する方法を説明します。
