# パーソナライゼーションフィールドを使用した電子メールの個人設定

パーソナライゼーションフィールドは、配信されるメッセージのコンテンツをパーソナライズする第 1 レベルの手段として使用されます。メインコンテンツに挿入するフィールドは、選択したデータソースのデータを挿入する位置を示します。

個人設定フィールドを件名行と電子メール配信のコンテンツに追加する方法について説明します。
